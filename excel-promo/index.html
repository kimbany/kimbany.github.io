엑셀 행사 처리기

엑셀을 업로드하면 미리 설정한 행사(1+1, 사은품 증정)에 따라 자동으로 가공해 줍니다.

1. 1+1 행사

아래 목록에 등록된 제품코드는 엑셀의 구매수량이 자동으로 ×2 처리됩니다. 적용 기간은 각 항목마다 따로 설정되며, 비워두면 상시 적용됩니다.

제품코드적용 기간관리

2. 사은품 증정 행사 (특정 제품코드)

대상 제품코드, 랜덤 사은품 코드들, 적용 기간, 수량 모드를 등록합니다. 해당 제품 구매 시 바로 아래 행에 사은품 행이 추가됩니다.

대상 제품코드사은품 풀(랜덤)수량 모드적용 기간관리

3. 모든 주문에 사은품 (전체 행사)

제품코드와 무관하게 모든 주문에 사은품을 증정합니다. 주문번호(A 컬럼)를 기준으로 묶어서, 같은 주문의 마지막 행 바로 아래에 사은품 행이 추가됩니다. 메뉴2의 "주문번호 컬럼" 매핑이 필요합니다.

사은품 풀(랜덤)수량 모드적용 기간관리

4. 설정 저장/관리

설정은 이 브라우저에 자동 저장됩니다(localStorage).

5. 클라우드 실시간 동기화

행사 설정이 자동으로 클라우드에 저장되고, 다른 PC/브라우저에서도 변경사항이 실시간으로 반영됩니다. 별도 설정 없이 바로 사용하세요.

1. 엑셀 업로드

선택된 파일 없음

2. 컬럼 매핑 (필수)

행사 적용에 필요한 컬럼을 지정해 주세요. 제품코드는 두 컬럼을 모두 지정할 수 있고, 우선 컬럼에 값이 있으면 그 값을, 비어있으면 보조 컬럼의 값을 사용해 매칭합니다. (예: 자체품목코드를 우선, 자체 상품코드를 보조)

3. 사은품 행에 복사할 컬럼

사은품 행이 추가될 때, 원본 주문 행에서 아래 체크된 컬럼들의 값이 그대로 복사됩니다. (제품코드/수량은 자동 처리)

4. 미리보기

5. 처리 & 다운로드

사용 흐름

  1. 메뉴 1에서 1+1 대상 제품코드와 사은품 행사 규칙을 등록합니다.
  2. 메뉴 2에서 주문 엑셀을 업로드하고, 컬럼 매핑을 확인합니다.
  3. “행사 적용해서 다운로드”를 누르면 가공된 엑셀이 받아집니다.

처리 규칙

  • 1+1: 등록된 제품코드 행의 구매수량 × 2 (변경된 수량 셀은 노란색)
  • 사은품 증정 (특정 코드): 등록된 제품코드 행 바로 아래에 사은품 행 삽입. 수량 모드:
    • 주문 1건당 1개: 매칭된 행마다 사은품 1개
    • 구매 수량만큼: 구매 수량 N개당 사은품 N개 (각 행 수량 1, 매번 랜덤 재추첨)
    메뉴2 "사은품 행에 복사할 컬럼"의 체크 컬럼이 원본에서 복사됨. 추가된 행 셀은 연두색.
  • 사은품 증정 (전체 주문): 주문번호로 묶어 같은 주문의 마지막 행 바로 아래에 사은품 행 추가. 수량 모드:
    • 주문 1건당 1개
    • 주문의 총 수량만큼: 그 주문 모든 행의 수량 합만큼
  • 적용 기간: 시작·종료일 비워두면 상시. 오늘 날짜가 범위 안일 때만 적용.

참고

  • 모든 처리는 브라우저 안에서만 이루어집니다. 파일이 외부로 전송되지 않습니다.
  • 원본 엑셀의 셀 수식이 있는 경우 값으로 평가된 결과가 사용됩니다.
