📥 블로그 / 인스타 미디어 다운로더

URL을 붙여넣으면 페이지의 이미지 · GIF · 영상을 한 번에 받아줍니다.
⚠️ 이 사이트(네이버/인스타 등)는 자동 스캔을 거의 다 막아놨어요.
대신 북마클릿(즐겨찾기 버튼)을 쓰면 100% 됩니다. 아래 두 가지 방법 중 하나 고르세요:
방법 A. 아래 빨간 버튼을 마우스로 잡고 → 브라우저 북마크 바로 드래그:
📌 미디어 추출기 (드래그)
방법 B. 드래그가 안 되면 → 누르고, 북마크 새로 만들 때 URL 자리에 붙여넣기.
→ 그 다음 받고 싶은 블로그 글을 직접 켜놓고 방금 만든 북마크를 클릭하면 새 창에 미디어가 다 떠요. 우클릭 저장.
🔖 인스타/네이버처럼 차단된 사이트 받는 법 (북마클릿)

아래 버튼을 북마크 바로 드래그해서 등록하세요. 그 다음 인스타/네이버 블로그 글을 켜놓고 북마크를 누르면, 그 페이지의 이미지·영상 링크가 새 창에 뜹니다. 거기서 우클릭 → "다른 이름으로 저장"으로 받으세요.

📌 미디어 추출기 (드래그)
  • 인스타: 게시물 하나의 페이지(/p/... 또는 /reel/...)에서 사용
  • 네이버 블로그: 글 본문 안(PostView iframe)에서 사용하면 더 잘 잡힘
  • 로그인이 필요한 비공개 게시물도 본인이 로그인된 상태면 동작합니다
💡 어떤 버튼을 눌러야 하나요?
  • 📂 폴더 선택 후 모두 다운로드 — 가장 편함. 자동으로 폴더에 저장. (CORS 차단 사이트는 안 됨)
  • ⬇ 전부 그냥 다운로드 — 기본 다운로드 폴더로. (역시 CORS 영향)
  • 🪟 새 탭에서 모두 열기 — 개발자도구에서 URL 클릭하는 것과 동일. 차단 안 당함. 각 탭에서 우클릭 → 저장. (팝업 차단 허용 필요)
  • 🔗 링크 리스트 모드 — 미리보기 대신 클릭 가능한 링크로 표시. 각 링크 우클릭 → "다른 이름으로 링크 저장". 차단 안 당함. 가장 확실한 방법.
  • 📋 URL 전체 복사 — URL 목록을 클립보드에. 다운로드 매니저(JDownloader 등)에 붙여넣기 좋음.
  • 미리보기 이미지 자체도 우클릭 → "이미지 저장" 으로 받을 수 있어요.
ℹ️ 안내 / 한계
  • URL 입력 방식은 CORS 프록시를 거치기 때문에 일부 사이트는 차단될 수 있어요. 그럴 땐 북마클릿이나 "새 탭 열기" / "링크 리스트" 모드를 쓰세요.
  • "폴더 선택 저장"은 크롬·엣지에서만 됩니다.
  • 저작권이 있는 콘텐츠는 개인 보관 외 용도로 재배포하지 마세요.
